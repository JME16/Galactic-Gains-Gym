# Galactic Gains Gym (GGG) Website Rollout Project

This repository documents the planning, coordination, and project management deliverables for the simulated rollout of the **Galactic Gains Gym (GGG)** website. The project serves as a real-world application of project management principles using CompTIA Project+ and PMBOK 7 guidelines.

## 📌 Project Overview

**Galactic Gains Gym (GGG)** is a multi-department health and wellness facility themed with characters from Marvel, DC, and Star Wars. This project simulates a full website rollout across departments, including:
- Aquatics
- Fitness & Strength Training
- Mindfulness
- Self-Defense
- Front Desk & Operations
- Pro Shop & Smoothie Bar

The project documents were created as part of a career transition into formal project management, highlighting leadership, scheduling, stakeholder coordination, and scope control.
